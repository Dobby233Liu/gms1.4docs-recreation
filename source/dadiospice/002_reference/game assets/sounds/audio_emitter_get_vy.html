audio_emitter_get_vy

Returns the current velocity along the y axis for a given emitter.

Syntax:

audio_emitter_get_vy(emitter);


Argument Description
emitter The index of the emitter to use.


Returns: Real


Description

This function returns the current velocity along the y axis for the given audio emitter.


Example:

if audio_emitter_get_vy(emitter_player) != 0
   {
   audio_emitter_velocity(emitter_player, 0, 0, 0);
   }

The above code checks the current y velocity of a given emitter and if it is not equal to 0, it is set to 0.
