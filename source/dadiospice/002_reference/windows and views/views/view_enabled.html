view_enabled

Whether views are enabled or not.

Syntax:

view_enabled


Returns: Boolean


Description

This variable controls whether any views that are visible within the room are enabled or not. If you have views visible and then disable this option, the whole room will be drawn to the screen scaled to the window size.


Example:

if !view_enabled
   {
   view_visible[0] = true;
   view_enabled = true;
   }

The above code checks to see if views are enabled for the room, and if they are not, it makes sure that view[0] is visible and then enables views for the room.
