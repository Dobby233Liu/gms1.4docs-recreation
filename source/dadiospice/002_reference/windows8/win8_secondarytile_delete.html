win8_secondarytile_delete

Remove a secondary Live Tile from the Windows 8 UI.

Syntax:

win8_secondarytile_delete(tile_id);


Argument Description
tile_id The unique identifying string of the tile to use.


Returns: N/A


Description

This function can be used to remove a secondary Live Tile from the Windows 8 UI. You must provide the unique tile id string, which is defined when you pin the tile to the UI using win8_secondarytile_pin function.


Example:

win8_secondarytile_delete("StoreTile");

The above code will remove the secondary Live Tile with the id "StoreTile".
