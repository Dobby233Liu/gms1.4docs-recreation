winphone_tile_back_image_wide

Add an image to the games wide Flip Live Tile "front".

Syntax:

winphone_tile_back_image_wide(img);


Argument Description
type The local path to the image to display.


Returns: N/A


Description

This function will take an image and show it on the "back" of the wide version of the Flip type Live Tile. This image must be an included file and not a sprite or a background, and if it is too small or too big for the tile it will be stretched or cropped to fit.


Example:

var tile;
tile = choose("tile1.png", "tile2.png", "tile3.png");
winphone_tile_back_image_wide(tile);

The above code will set the "back" of the wide Live Tile to be any one of three included files.
