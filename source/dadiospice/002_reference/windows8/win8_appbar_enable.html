win8_appbar_enable

Enable the App Bar for the game.

Syntax:

win8_appbar_enable(enabled);


Argument Description
enabled Set to true to enable the App Bar and false to disable.


Returns: N/A


Description

This function can be used to switch on (true) or off (false) the App Bar for your game. The App Bar is accessed by clicking the Right Mouse Button while in your game and, if it is on, then you can add your own custom buttons to it. If this is set to off, then the App Bar is disabled (the default value is false).


Example:

win8_appbar_enable(true);
win8_appbar_add_element("button", "mute", "Mute/Unmute Music", "selection", "Toggle the music on or off", scr_MusicSwitch);

The above code will switch on the App Bar, then add a button to it to mute the music.
