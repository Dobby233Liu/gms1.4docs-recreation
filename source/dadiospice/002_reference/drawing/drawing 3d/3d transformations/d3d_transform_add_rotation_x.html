d3d_transform_add_rotation_x

Adds a rotation around the x-axis with the indicated amount.

Syntax:

d3d_transform_add_rotation_x(angle)


Argument Description
angle The angle to rotate around the x-axis.


Returns: N/A


Description

This function will rotate the object around the x-axis by the specified amount (in degrees), relative to any previously performed transforms.


Example:

var tex = background_get_texture(bck_Wall); d3d_transform_set_identity();
d3d_transform_add_rotation_x(90);
d3d_draw_block(-50, -50, -50, 50, 50, 50, tex, 100, 100);
d3d_transform_set_identity();

The above code tells GameMaker: Studio to rotate the block 90 degrees along the x-axis (in addition to any previous transforms) before drawing it.
