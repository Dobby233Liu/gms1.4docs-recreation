d3d_transform_stack_clear

Clears the stack of transformations.

Syntax:

d3d_transform_stack_clear()


Returns: N/A


Description

This will clear the stack of all transforms. In GameMaker: Studio there is only one, global, transform stack and it is used by all instances for all transforms. This means you should be careful to clear the stack when finished for each instance with this command otherwise you will get a stack overflow error.


Example:

d3d_transform_stack_clear();

The above code will clear the transformation stack.
