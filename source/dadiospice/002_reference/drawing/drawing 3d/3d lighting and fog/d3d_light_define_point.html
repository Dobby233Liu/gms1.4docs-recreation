d3d_light_define_point

Defines a point light.

Syntax:

d3d_light_define_point(ind, x, y, z, range, col)


Argument Description
ind The index number of the light.
x The x position of the light.
y The y position of the light.
z The z position of the light.
range The light range.
col The colour of the light.


Returns: N/A


Description

To use lighting you obviously need to define lights. In GameMaker: Studio two different lights exist: directional lights (like the sun), and positional lights. This function is for a positional light, where you can define the x, y and z position of the light, the light range and the colour of the light (which will also affect the perceived intensity of the light as certain colours appear "darker" than others). You must also give the light an index number which what will be used in other functions to reference it.

Note: There are only 8 hardware lights available, so you should use these functions to define and manage "active" lights, not all lights in the scene.


Example:

d3d_set_lighting(true);
d3d_light_define_point(1, 200, 123, 50, 2000, c_white);
d3d_light_enable(1, true);

The above code will enable lighting for the whole scene, then define a white light at a specific point in the room space, and finally turn that light on.
