ds_list_destroy

Destroys a given list and removes it from memory.

Syntax:

ds_list_destroy(id);


Argument Description
id The id of the data structure to remove.


Returns: N/A


Description

This function will remove the given list data-structure from memory, freeing up the resources it was using and removing all values that it contained. This function should always be used when you are finished using the ds_list to prevent memory leaks that can slow down and crash your game.


Example:

if lives = 0
   {
   ds_list_destroy(AI_list);
   room_goto(rm_Menu);
   }

The above code will check the value of the built in global variable "lives" and if it is 0, it destroys the ds_list indexed in the variable "AI_list" and then changes rooms.
