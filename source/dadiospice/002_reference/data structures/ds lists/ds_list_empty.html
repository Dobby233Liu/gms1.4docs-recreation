ds_list_empty

Returns whether the given list is empty or not.

Syntax:

ds_list_empty(id);


Argument Description
id The id of the data structure to check.


Returns: Boolean


Description

With this function you can check the given ds_list to see if it is empty (returns true) or not (returns false).


Example:

if count == 15 && !ds_list_empty(command_list)
   {
   ds_list_clear(command_list);
   alarm[0] = room_speed;
   count = 0;
   }

The above code checks a variable to see if it has reached a specific value and if it has it clears the ds_list indexed in the variable "command_list", sets an alarm, and resets the variable to 0.
