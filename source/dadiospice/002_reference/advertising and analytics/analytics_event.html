analytics_event

Send an analytics event to your analytics provider.

Syntax:

analytics_event(string)


Argument Description
string A string to send to the provider.


Returns: N/A


Description

This function will send the specified text to the analytics provider that you have set up through the Global Game Settings - Analytics Tab. This function can be used to create a custom analytic to track something outside of the scope of the provider being used.


Example:

if hs_new
   {
   analytics_event("New hiscore of " + string (score));
   }

The above code will check a variable to see if it is true and if it is then a special analytics event will be triggered with the specified string.
