The Create Event

The create event is one of the many events that can be triggered in an instance and may contain code or actions.

The Create event
This event happens when an instance of the object is first created, and is the very first thing that happens within an instance placed in the room through the room editor when a room is entered. This means that this event is the ideal place to initialize variables, start Timelines, set paths etc... and do anything else that generally only needs to be done once or only when an instance first appears in the room.

NOTE: This event will always be called the moment an object is created.
